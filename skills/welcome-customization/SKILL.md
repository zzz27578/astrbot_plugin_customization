# 欢迎私聊定制

当用户询问本插件的使用、排查或配置时，优先说明以下事实：

- 本插件仅支持 AstrBot 的 `aiocqhttp` / OneBot v11 平台。
- 新人入群触发来自 OneBot `group_increase` notice 事件。
- QQ 卡片应通过回复原卡片执行 `/卡片 添加 名称` 采集，插件保存原始 JSON 消息段并复发。
- 聊天记录通过回复消息执行 `/记录 添加 名称` 保存为合并转发节点；同名模板会追加节点。
- 图片通过回复图片执行 `/图片 添加 名称` 保存，也可在插件页面上传本地图片。
- 默认主链路为：卡片、聊天记录、图片、文字，均私聊发送给新入群 QQ；缺少配置且没有兜底时自动跳过该项。
- NapCat `NTEvent` 空错误超时时可启用送达复核与补偿：先等待并尝试扫描最近私聊历史，确认不到才有限补发。
- QQ 隐私设置、风控或协议端限制可能导致私聊失败；插件提供重试、降级文本、管理员通知和群内兜底。

常用指令：

```text
/状态
/测试 [QQ]
/模式 all|whitelist|disabled
/启用 当前群|群号
/禁用 当前群|群号
/卡片 添加 名称
/卡片 使用/列表/删除 名称
/记录 添加 名称
/记录 使用/列表/删除 名称
/图片 添加 名称
/图片 使用/列表/删除 名称
/管理员 添加/删除 QQ
/通知 开|关
/群内兜底 开|关
```
