# 欢迎私聊定制

当用户询问本插件的使用、排查或配置时，优先说明以下事实：

- 本插件仅支持 AstrBot 的 `aiocqhttp` / OneBot v11 平台。
- 新人入群触发来自 OneBot `group_increase` notice 事件。
- QQ 卡片应通过回复原卡片执行 `/欢迎 卡片 添加 名称` 采集，插件保存原始 JSON 消息段并复发。
- 聊天记录通过回复消息执行 `/欢迎 记录 添加 名称` 保存为合并转发节点；同名模板会追加节点。
- 图片通过回复图片执行 `/欢迎 图片 添加 名称` 保存，也可在插件页面上传本地图片。
- 默认主链路为：卡片、聊天记录、图片，均私聊发送给新入群 QQ。
- QQ 隐私设置、风控或协议端限制可能导致私聊失败；插件提供重试、降级文本、管理员通知和群内兜底。

常用指令：

```text
/欢迎 状态
/欢迎 测试 [QQ]
/欢迎 模式 all|whitelist|disabled
/欢迎 启用 当前群|群号
/欢迎 禁用 当前群|群号
/欢迎 卡片 添加 名称
/欢迎 卡片 使用/列表/删除 名称
/欢迎 记录 添加 名称
/欢迎 记录 使用/列表/删除 名称
/欢迎 图片 添加 名称
/欢迎 图片 使用/列表/删除 名称
/欢迎 管理员 添加/删除 QQ
/欢迎 通知 开|关
/欢迎 群内兜底 开|关
```
